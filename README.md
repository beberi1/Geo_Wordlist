## рЃЏрЃЮрЃЊрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљ

Names рЃЊрЃљ Surnames рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃАрЃ«рЃЋрЃљ рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃњрЃЋрЃљрЃарЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ.
рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃарЃЮрЃЏ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў рЃўрЃДрЃЮрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃю рЃњрЃЋрЃљрЃарЃў.

рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃЏрЃЮрЃЊрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљ рЃўрЃАрЃћ рЃарЃЮрЃЏ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃЮрЃА рЃЎрЃўрЃЊрЃћрЃЋ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ. рЃерЃћрЃЋрЃћрЃфрЃљрЃЊрЃћ рЃЉрЃћрЃЋрЃарЃў рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃЊрЃљрЃЏрЃћрЃарЃЌрЃЮ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ (рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ рЃўрЃДрЃЮрЃА)

рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃљрЃЏрЃЮрЃЊрЃўрЃцрЃўрЃфрЃўрЃарЃЮ рЃгрЃћрЃарЃўрЃљ рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ ­ЪЉЄ "рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃЊрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљрЃерЃў" рЃњрЃарЃљрЃцрЃљрЃерЃў

## рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃў NS_Generator_Mixer

* рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃљрЃерЃџрЃљ - рЃ«рЃљрЃќрЃў 225

рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃљ рЃљрЃю рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊ рЃгрЃљрЃерЃџрЃљ рЃЌрЃБ рЃњрЃАрЃБрЃарЃА рЃарЃЮрЃЏ рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃа рЃгрЃљрЃўрЃерЃљрЃџрЃЮрЃА
